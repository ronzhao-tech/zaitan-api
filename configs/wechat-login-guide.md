# 微信登录申请指南

## 申请条件
- 企业资质（营业执照）
- 已上线的网站或移动应用
- 个人开发者较难申请，建议先用手机号登录

## 申请步骤

### 1. 注册微信开放平台
访问: https://open.weixin.qq.com

### 2. 企业认证
- 填写企业信息
- 上传营业执照
- 支付 ¥300 认证费（一次性）

### 3. 创建应用
1. 管理中心 → 创建移动应用 / 网站应用
2. 填写应用信息:
   - 应用名称: 再探
   - 应用简介: AI智能稍后阅读工具
   - 应用官网: https://ronzhao-tech.github.io/zaitan-h5
3. 上传应用图标（28x28, 108x108）

### 4. 配置授权回调域
1. 应用详情 → 修改
2. 授权回调域: ronzhao-tech.github.io
3. 等待审核（1-7个工作日）

### 5. 获取 AppID 和 AppSecret
审核通过后，在应用详情页查看

## 替代方案
如果暂时无法申请微信登录，可以:
1. 使用手机号 + 验证码登录（已实现）
2. 使用 Supabase Auth 的魔法链接登录
3. 后续再补充微信登录
