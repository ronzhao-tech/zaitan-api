## Phase 2: 后端部署（30分钟）

### ✅ 2.1 配置数据库
```bash
cd zaitan-api
npx prisma db push
npx prisma generate
```

### ✅ 2.2 部署到 Vercel
```bash
npm i -g vercel
vercel --prod
```

### ✅ 2.3 更新前端 API 地址
修改 `zaitan-h5/index.html` 中的 API 地址为 Vercel 部署地址
