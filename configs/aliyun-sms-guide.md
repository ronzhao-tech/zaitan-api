# 阿里云短信服务申请指南

## 申请条件
- 个人/企业实名认证
- 已备案的网站或应用（个人可用App）

## 申请步骤

### 1. 注册阿里云账号
访问: https://www.aliyun.com

### 2. 实名认证
- 个人认证: 身份证 + 人脸识别
- 企业认证: 营业执照 + 法人身份证

### 3. 开通短信服务
1. 访问: https://www.aliyun.com/product/sms
2. 点击「立即开通」
3. 按量付费，无最低消费

### 4. 申请短信签名
1. 进入短信服务控制台
2. 左侧菜单: 国内消息 → 签名管理
3. 点击「添加签名」
4. 填写:
   - 签名: 再探（或你的品牌名）
   - 适用场景: 验证码
   - 证明类型: 企业营业执照 / App应用
5. 等待审核（通常1-2小时）

### 5. 申请短信模板
1. 左侧菜单: 国内消息 → 模板管理
2. 点击「添加模板」
3. 填写:
   - 模板类型: 验证码
   - 模板名称: 登录验证码
   - 模板内容: 您的验证码是${code}，5分钟内有效。
4. 等待审核

### 6. 获取 AccessKey
1. 右上角头像 → AccessKey 管理
2. 创建 AccessKey
3. 保存 AccessKey ID 和 Secret

### 7. 费用说明
- 验证码短信: ¥0.045/条
- 新用户有免费试用额度

## 测试建议
开发阶段可以使用测试验证码（如 000000），上线前再接入真实短信。
